## Usage

**TEMPORARY**

Run the scripts in this order:

1. `data_praw.py`
2. `data_processing.py`
3. `find_useful_features.py`

### Prerequisites

- `spacy`
- `sklearn`
- `scipy`
- `numpy`
- `psaw`
- `pandas`
- `requests`
- `matplotlib`

Download the English Spacy model with
```sh
$ python -m spacy download en
```
